## Security Onion VM Setup

Security Onion ISO: [https://securityonionsolutions.com/software/](https://securityonionsolutions.com/software/)

I am on version 2.4.200

VM specs (recommended minimum):
- 16GB Memory
- 8 Processors
- 200GB Storage
- Network Adapters:
    - Adapter 1: SOC (10.10.3.0/24)
    - Adapter 2: DMZ

## Security Onion Web Interface + Testing

From the Ubuntu VM, I generated some failed SSH logins.

![29](/assets/img/cyberlab2/setup/so-setup/29.png){: width="800" height="400" }

![30](/assets/img/cyberlab2/setup/so-setup/30.png){: width="800" height="400" }
